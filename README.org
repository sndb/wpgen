=wpgen= is a random maze image generator. It writes the generated PNG
image to stdout (or a file, if stdout is a terminal). Its output can
then be transformed using =imagemagick= to create a nice background
for your desktop.

#+begin_src shell
$ wpgen
usage: wpgen -width SIZE -height SIZE [-fg COLOR] [-bg COLOR] [-step SIZE] [-x-offset SIZE] [-y-offset SIZE]

$ wpgen -fg 2f822f -bg 00422a -width 251 -height 199 -step 4 -x-offset 1 -y-offset 1 | magick - -magnify example.png
#+end_src
